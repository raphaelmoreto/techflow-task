# TechFlow Task Manager

## 📌 Objetivo
Sistema simples de gerenciamento de tarefas baseado em metodologias ágeis.

## 🚀 Funcionalidades
- Criar tarefas
- Listar tarefas
- Atualizar tarefas
- Deletar tarefas

## 🧠 Metodologia
Utilização de Kanban para organização:
- A Fazer
- Em Progresso
- Concluído

## 🛠 Tecnologias
- Python
- PyTest
- GitHub Actions

## ▶️ Como executar

```bash
pip install fastapi uvicorn
uvicorn app.main:app --reload
```

## 🔄 Gestão de Mudanças

Durante o desenvolvimento, o cliente solicitou a adição de prioridade nas tarefas,
para melhorar a organização e tomada de decisão da equipe.
